# Random Phone Number Generator
This generates random phone numbers

## Setup/installation

#### Requirement
 - Node

#### How to install
```
$ yarn install
$ yarn start
```

#### To test
```
$ yarn test
```

run `yarn test:watch` for automatic reload of tests when file changes.
